# Ninja-Traininggrounds
A small demo game for SocialBrothers

## Goal
Try to kill as many robots as you can

## Controls
<ul>
  <li>A/D to move</li>
  <li>Space to jump</li>
  <li>LeftShift to slide</li>
  <li>E to make a melee attack</li>
  <li>F to make a ranged attack</li>
</ul>
